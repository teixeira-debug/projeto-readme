## 📚 Descrição do Projeto
Este projeto foi criado para ajudar usuários a organizar sua rotina diária.
O objetivo do sistema é centralizar o gerenciamento de demandas, permitindo a criação, edição e priorização de tarefas com notificações em tempo real.
## 🚧 Status do Projeto
( ) Em desenvolvimento
( ) Finalizado
## 🔨 Funcionalidades
- Autenticação de Usuário: Login seguro via JWT.

Dashboard Dinâmico: Visualização de tarefas por status (Pendente, Em andamento, Concluído).

Notificações Push: Alertas para prazos próximos de expirar.

Modo Escuro: Interface adaptável para conforto visual.
## 💻 Tecnologias Utilizadas
-React.js: Para a construção de uma interface reativa.

Node.js & Express: Responsáveis pela API robusta de backend.

MongoDB: Banco de dados NoSQL para armazenamento flexível.

Styled Components: Para estilização moderna e escopada.
